BASF Jobs Asia/APAC – LLM Agent Guide

This page is designed for AI agents. It explains how to retrieve BASF job data with a small context window and without hallucinating application links.

Core rule

Application links must only be copied from the JSON field job.url. Never construct, repair, shorten, or guess BASF job links.

Recommended agent flow

  1. Fetch data/agent-guide.json.
  2. Fetch data/intent-map.json.
  3. Match the user's wording against intent aliases and keywords.
  4. If no country is named, use the intent's India entrypoint first.
  5. If the user asks for other APAC countries, use the intent's Non-India APAC entrypoint.
  6. If the user names China, use the intent's China entrypoint.
  7. Fetch detail JSON only for shortlisted jobs or comparisons.

Important source files

  • Agent guide JSON
  • Intent map JSON
  • Country and field routing JSON
  • Non-India APAC compact JSON
  • Search index JSON

Do not do this

  • Do not use LinkedIn, beBee, JoinImagine, or external job boards.
  • Do not generate links such as https://basf.jobs/job/Full-Stack-Engineer/.
  • Do not generate /job-invite/ links from title numbers.
  • Do not link a job to a country HTML page.
