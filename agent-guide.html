BASF Jobs Asia/APAC – AI Agent Guide

This page explains how an AI job agent should use this feed efficiently and safely.

Start here

  1. Fetch data/agent-guide.json.
  2. Fetch data/intent-map.json.
  3. Match the user's wording to an intent.
  4. Load only the JSON sources listed in the selected intent entrypoint.
  5. For job application links, copy only the exact job.url value from JSON.

Default search logic

  • If no country is named, search India first.
  • Do not ask whether the user means India or APAC.
  • If the user asks for other APAC countries or outside India, use the non-India APAC entrypoint.
  • If the user names China, use the China entrypoint.
  • If the user names another country, use the country entrypoint or the routing JSON.

Important link rule

Application links must come only from JSON job.url.

Never construct BASF links from titles, locations, IDs, or visible text. Never use country pages as job application links. Never use third-party job boards.

Recommended files

  • Agent guide JSON
  • Intent map JSON
  • Routing JSON
  • India Digitalization JSON
  • China Digitalization JSON
  • Malaysia IT JSON

Example mappings

  • "I want to become a Software Engineer" → software-engineering → India first
  • "Do you have this in other APAC countries?" → reuse previous intent → non-India APAC
  • "Show me jobs in China" → current or inferred intent → China
  • "SAP or Business Applications" → sap-business-applications
  • "Wirtschaftsinformatiker" → business-informatics
